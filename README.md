# CI/CD Docker Project

This project contains a simple Node.js application, along with a `Dockerfile` for containerization and a `Jenkinsfile` for setting up a CI/CD pipeline.

## Project Structure

- `app/`: Contains the Node.js source code (`server.js`) and dependencies (`package.json`).
- `Dockerfile`: Instructions to build the Docker image for the app.
- `Jenkinsfile`: Defines the Jenkins pipeline stages (Build, Test, Deploy).
